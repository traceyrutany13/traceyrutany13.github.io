Corsair CO-9050106-WW iCUE QL140 RGB White Dual Fan Kit PCケースファン

商品特長・スペック

★ CORSAIR iCUE制御に対応した140mm 両面RGBファン
iQUE QL RGB WHITEは、CORSAIR iCUE制御に対応した 両面RGBファンです。裏表合わせて4つのRGBライトループは34個の高輝度LEDを備え、様々なライティングプロファイルの設定はもちろん、ファンの回転数、グループ同調設定などが可能です。独自の薄型ファンブレード、回転抵抗を低減させたハイドロリックベアリング、共振を抑制する防振ラバーダンパーにより、優れた静音性を実現します。ファン1個の増設用パッケージ、ファンとLighting Node COREを同梱したバリューパッケージ、用途で選べる2つのモデルと、120mm・140mmの2サイズをそれぞれラインアップ。

★ 裏表4つのRGBライトループに34個の高輝度LED
ファンブレード中心部分と、ハウジング部分にリング状のライトループを合計4対、ファンの両面に備えています。あらゆる方向に最大限のライティング効果を発揮し、色鮮やかにシステムをドレスアップします。

★ 最大6つのRGBファンを同時制御するiCUE Lighting Node CORE
iQUE QL RGB WHITEは、Lighting Node COREにより最大6つまでRGBライティングの同時制御が可能です。CORSAIR iCUEソフトウェア上から様々なライティングプロファイルの設定はもちろん、ファンの回転数、グループ同調設定などが思いのままに制御可能です。
※本製品のライティング機能にはLighting Node COREまたはLighting Node PRO(別売)が必要です。
初めて本製品をご使用される方はLighting Node CORE同梱のQL RGB Triple Fan Kitをお買い求めください。

★ ファン中央両面に刻印されたメタリックロゴ
ファン両面にメタリックロゴをあしらい、設置方向に関わらず高級感あるライティングを演出します。

★ PWM対応4ピン電源コネクタ搭載
PWM対応により、システム負荷に応じて最適な冷却パフォーマンスを発揮します。

★ 回転抵抗を低減させたハイドロリックベアリング
ファンの軸受け部分は回転抵抗を低減させたハイドロリックベアリングを採用しています。従来のボールベアリングタイプに比べ回転抵抗が低く、ノイズを抑えることで少ない電力でより静かに駆動可能です。

★ 共振を抑制する防振ラバーダンパー
ファンマウント部分には、共振を抑制する防振ラバーダンパーを備えています。フレーム接触面との間にクッション効果を持たせることで、不快な振動やノイズを抑制します。
