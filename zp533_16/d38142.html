説明:

The NanoPi NEO3 is a newly released super tiny ARM board by FriendlyElec. lt uses Rockchip"s 64-bit quad-core SoCRK3328(ARM Cortex-A53).These features make it especially suitable for applications that require high data throughput , speedy data transmission andhigh performance. Hobbyists and makers will just love it.

-Specification-

CPU Model: Rockchip RK3328

Number of Cores: Quad-Core Cortex-A53 64-bit

Frequency: 4 x Cortex-A53 Up to 1.3GHz

Memory RAM: 1GB 32-bit DDR4

Storage:MicroSD Slot: MicroSD x1 for external storage up to 128GB

Connectivity Ethernet: Gbps Ethernet x1 with unique MAC

USB: USB 3.0 Host x1: USB Type A; USB 2.0 Host x2: 2.54mm 4x2pin-header; Type-C x1: Power Input Only

Pin-header: GPIO: 2.54mm pitch 26pin-header, include I2C, UART, SPI, I2S, GPIO;

Serial Debug Port x1: 2.54mm pitch 3pin-header, 115200bps

LED: PWR LED(Red) x1; STAT LED(Green) x1

Button: User button x1

Working Temperature -20℃ to 70℃

Power: 5V/1A, via Type-C or GPIO

-OS Support-

FriendlyWrt 21.02.1 (64-bit)

FriendlyCore Focal Lite (Base on Ubuntu 20.04)

-Dimension-

PCB: 6 Layer

Weight: 22g

Size: 48x48x10 mm

-Package Content-

1x Nanopi NEO3 1GB Mini Router Single Board- (with unique MAC Chip)
